\# Pablitus Prompt Framework (PPF)



An intelligent PowerShell prompt built on top of Oh My Posh.



\## Features



\- Intelligent project detection

\- Git integration

\- Python detection

\- .NET detection

\- Node.js detection

\- Docker support

\- Kubernetes support

\- WSL awareness

\- SSH awareness

\- Lightweight

\- Fully customizable
